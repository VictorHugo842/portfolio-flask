## Como Executar Localmente

1. Clone o repositório:
    ```bash
    git clone https://github.com/VictorHugo842/portfolio-flask
    ```
2. Instale as dependências:
    ```bash
    pip install -r requirements.txt
    ```
3. Execute a aplicação:
    ```bash
    flask run
    ```
